Reverse Linked List II

创建时间：2014/6/20 14:09
更新时间：2014/6/20 14:09

Reverse a linked list from position m to n. Do it in-place and in one-pass.

For example:
Given 1->2->3->4->5->NULL, m = 2 and n = 4,

return 1->4->3->2->5->NULL.

Note:
Given m, n satisfy the following condition:
1 ≤ m ≤ n ≤ length of list.

/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode(int x) {
 *         val = x;
 *         next = null;
 *     }
 * }
 */
public class Solution {
    public ListNode reverseBetween (ListNode head, int m, int n) {
        if(head == null )
             return null ;
        if(m == n)
             return head;
        ListNode pointer = new ListNode(0 ),leftTail,pre,temp ,p;
        pointer .next = head;
        int count = 0;
        leftTail = pointer;
        while(count < m - 1 ){
            leftTail = leftTail.next;
            count ++;
        }
        pre = null ;
        count ++;
        p = leftTail.next;
        while(count <= n){
            temp = p. next;
            p .next = pre;
            pre = p;
            p = temp;
            count ++;
        }
        leftTail .next. next = p ;
        leftTail .next = pre;
            
        return pointer.next;
    }
}
