Gas Station

创建时间：2014/6/21 21:03
更新时间：2014/6/21 21:03

There are N gas stations along a circular route, where the amount of gas at station i is gas[i].

You have a car with an unlimited gas tank and it costs cost[i] of gas to travel from station i to its next station (i+1). You begin the journey with an empty tank at one of the gas stations.

Return the starting gas station's index if you can travel around the circuit once, otherwise return -1.

Note:
The solution is guaranteed to be unique.

public class Solution {
    public int canCompleteCircuit (int[] gas, int[] cost ) {
         if ( null == gas || gas .length == 0 || cost == null || cost.length == 0)
             return - 1;
         int tank, length = gas .length;
         int[] sliceRemain = new int [length];
         boolean[] invalid = new boolean [length];
         for ( int i = 0 ; i < length; i ++)
            sliceRemain [i] = gas[i ] - cost[i ];
         for ( int start = 0; !invalid[start ];) {
             int count = 0;
            tank = 0 ;
             for ( int i = start; count < length; count++) {
                tank += sliceRemain[i];
                 if ( tank >= 0) {
                     if ( count == length - 1 )
                         return start;
                    i = i != length - 1 ? i + 1 : 0;
                 } else {
                     if ( count == length - 1 )
                         return - 1;
                     else {
                        invalid [start] = true;
                        start = i != length - 1 ? i + 1 : 0;
                         break;
                     }
                 }
             }
         }
         return - 1;
    }
}
